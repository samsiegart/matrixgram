### Controls
- j: next
- k: prev
- ctrl+c: exit

----------------------------------------------------------------------

### Setup

#### OS X:
 - Python: `brew install python`
 - OpenCV: `brew tap homebrew/science`, `brew install opencv`
 - Requirements: `pip install -r requirements.txt`
 - Launch: `python matrix.py`
